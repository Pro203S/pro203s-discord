# CLI 사용하기

이 문서는 [@pro203s/discord](https://npmjs.com/package/@pro203s/discord)를 사용할 때를 가정한 문서입니다.  

프로젝트 세팅이 되어있지 않다면 [시작하기](./gettingStarted.md)를 참고해주세요!  

## 1. 봇 실행하기

프로젝트의 봇을 실행하려면 `npx @pro203s/discord start`를 입력하면 봇이 실행됩니다.  

예시 출력:
```
@pro203s/discord v1.0.0
✓ Ready in 1317ms
  Logged In as <봇 이름>#<봇 태그>
```

## 2. 모든 커맨드 삭제

봇의 모든 애플리케이션 명령어를 삭제하려면 `npx @pro203s/discord clear-commands`를 입력해주세요.  

예시 출력:
```
✔ Cleared!
```
